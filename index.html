Introduction

This course will teach you how to use the Photopea online image editing program to do basic image processing tasks such as: cropping, resizing, colour balancing and sharpening. Also more advanced creative tasks such as: selecting and working with parts of images, using layers (blending) and other enhancement techniques, and making an image by combining images, will be covered.

You will need to have basic skills in using a web browser and saving files. Both Windows, Macintosh and Linux based computers can be used as long as they have an up to date version of a web browser such as Chrome or Firefox.

If you choose to use your own laptop it will need to have a mouse as it is needed to make accurate selections.

There will be two sessions, each of two hours.

Session One

The following will be covered in this session:

  1. Introduction to the course website (it contains instructions information and sample images to work with)
  2. What is an image: resolution, different image formats (jpeg, png, gif, bitmap)
  3. Introduction to Photopea and its interface
  4. Cropping, resizing, colour balancing, sharpening and saving an image
  5. The Text tool

Session Two

The following will be covered in this session:

  1. A recap and practice of the basic editing techniques covered in session one
  2. Using the selection tools
  3. Layers: blending modes and adjustments
  4. Combining Images
