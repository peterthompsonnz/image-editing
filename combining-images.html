Combining Images

Once you know how to use the selection tools and create and manipulate layers it is quite easy to combine parts of images. These are the steps to follow:

  1. Open the images you want to work with. They will appear as separate tabs in Photopea. There can be more than two images. One image will be the target image where parts of other images are added as new layers, and the others will be used for sources for these parts.
  2. Using the source images, select the parts you want to use using a selection tool. Copy the selections to the computers clipboard (Edit->Copy or Ctrl+c), switch to the tab that contains the target image and paste from the clipboard (Edit->Paste or Ctrl+v). Move, erase, transform or edit the new layer(s) as needed.
  3. Save the composite image.
